# OSSAfrica Swag Designs

This directory contains designs for merchandise, swag, and promotional items.

## Types of Swag

- **Apparel**: T-shirts, hoodies, hats
- **Stickers**: Laptop stickers, die-cut stickers
- **Accessories**: Bags, water bottles, notebooks
- **Office**: Mugs, pens, mousepads
- **Tech**: USB drives, phone accessories
- **Event Items**: Badges, lanyards, wristbands

## Design Specifications

### T-Shirts

#### Print Areas
- **Front**: 12" x 14" maximum
- **Back**: 14" x 16" maximum
- **Pocket**: 4" x 4" maximum
- **Sleeve**: 3" x 3" maximum

#### Print Types
- **Screen Print**: Simple designs, solid colors
- **DTG (Direct to Garment)**: Complex designs, full color
- **Vinyl**: Single color, durable

#### File Requirements
- **Format**: Vector (AI, EPS, PDF) or high-res PNG
- **Resolution**: 300 DPI minimum
- **Color Mode**: Spot colors for screen print, CMYK for DTG
- **Colors**: Limit to 4-6 for screen printing

### Stickers

#### Sizes
- **Small**: 2" x 2" to 3" x 3"
- **Medium**: 4" x 4" to 5" x 5"
- **Large**: 6" x 6" and up

#### Types
- **Die-cut**: Custom shapes, with or without border
- **Kiss-cut**: Custom shape on backing sheet
- **Standard**: Square, circle, or rectangular

#### File Requirements
- **Format**: Vector (AI, EPS, SVG) preferred
- **Resolution**: 300 DPI minimum for raster
- **Bleed**: Add 0.125" bleed for die-cut
- **Colors**: CMYK or Pantone

### Accessories

#### Bags (Tote/Drawstring)
- **Print Area**: Varies by bag size
- **Common**: 10" x 10" to 14" x 14"
- **Colors**: 1-2 colors typical for screen print

#### Mugs
- **Print Area**: 8.5" x 3.5" (wrap around)
- **Resolution**: 300 DPI
- **Format**: Unwrapped design template

#### Notebooks
- **Cover**: 5" x 7" or 8.5" x 11"
- **Resolution**: 300 DPI
- **Bleed**: Add 0.125" bleed

## Design Guidelines

### General Principles
- **Simple is Better**: Designs should work at small sizes
- **High Contrast**: Ensure visibility on product
- **Brand Consistency**: Follow brand guidelines
- **Scalability**: Design should work at various sizes

### Color Considerations
- **Limit Colors**: Fewer colors reduce printing costs
- **Test on Product**: Consider product color/material
- **Print vs Digital**: Colors may appear different printed
- **Pantone Matching**: Use Pantone colors for consistency

### Typography
- **Readable**: Use clear, legible fonts
- **Size**: Minimum 8pt for printed text
- **Weight**: Medium to bold for better printing
- **Licensing**: Ensure fonts are licensed for commercial use

## Production Files

### Required Files for Production
1. **Source File**: Editable design (AI, PSD, etc.)
2. **Vector File**: Scalable format (AI, EPS, PDF, SVG)
3. **Color Specs**: Pantone or CMYK values
4. **Mock-up**: Visualization on product
5. **Instructions**: Any special printing notes

### Mock-ups
- Show design on actual product
- Multiple angles if applicable
- Help visualize final result
- Use for approval before production

## Print-Ready Guidelines

### Color Modes
- **Screen Print**: Spot colors (Pantone)
- **Digital Print**: CMYK
- **Embroidery**: Thread colors (closest Pantone match)

### Resolution
- **Vector**: Preferred for all designs
- **Raster**: 300 DPI minimum at actual size
- **Large Format**: May use 150 DPI at size

### Fonts
- **Outline**: Convert text to outlines/paths
- **Embed**: Ensure fonts are embedded
- **License**: Verify commercial use license

### Bleed and Safe Area
- **Bleed**: Extend 0.125" beyond trim edge
- **Safe Area**: Keep important elements 0.25" from edge
- **Guidelines**: Include trim and safe area guides

## Contributing

When contributing swag designs:

1. **Research**: Verify product specifications
2. **Mockup**: Create realistic product mockup
3. **Files**: Provide all required file formats
4. **Colors**: Specify exact color values
5. **Instructions**: Include printing/production notes
6. **License**: Ensure all elements are properly licensed
7. **Test**: Review design at actual printed size
8. **Document**: Include material and vendor suggestions

### Submission Checklist
- [ ] Design follows brand guidelines
- [ ] Vector files provided
- [ ] Colors specified (Pantone/CMYK)
- [ ] High-resolution mockups included
- [ ] Bleed and safe areas marked
- [ ] Fonts outlined or embedded
- [ ] Production specifications documented
- [ ] Multiple format options (if applicable)
- [ ] File naming convention followed

## Vendors and Production

When ready for production:
- Research reputable vendors
- Request samples when possible
- Verify minimum order quantities
- Check turnaround times
- Consider cost vs. quality
- Review proofs carefully before approving

## Budget Considerations

- Screen printing: Best for large quantities, fewer colors
- DTG: Good for small batches, full color
- Embroidery: Premium look, higher cost
- Stickers: Affordable, various options
- Always get quotes from multiple vendors
