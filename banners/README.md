# OSSAfrica Banners

This directory contains banner images for websites, events, and promotional materials.

## Types of Banners

- **Website Headers**: Hero banners for websites
- **Event Banners**: Graphics for conferences, meetups, and events
- **GitHub Banners**: Repository and organization headers
- **Email Banners**: Headers for newsletters and email campaigns

## Standard Dimensions

### Website Banners
- **Hero Banner**: 1920 x 600 px (full width)
- **Section Banner**: 1200 x 400 px

### GitHub
- **Organization Profile**: 1280 x 640 px
- **Repository Social Preview**: 1280 x 640 px

### General
- **Wide Banner**: 1200 x 300 px
- **Standard Banner**: 1000 x 400 px

## Design Guidelines

### Content
- Keep text minimal and readable
- Use high contrast between text and background
- Ensure key information is visible at thumbnail size

### Format
- Save as PNG or JPG
- Optimize file size for web (target < 500KB)
- Provide high-resolution versions for print when applicable

### Responsive Considerations
- Important content should be centered (safe zone: center 1000px)
- Avoid placing critical elements at edges
- Test appearance at different viewport sizes

## File Naming Convention

```
[purpose]-banner-[dimensions]-[variation].[ext]

Examples:
- website-hero-banner-1920x600.png
- event-banner-conference-2024.jpg
- github-org-banner-1280x640.png
```

## Contributing

When contributing banners:
1. Use standard dimensions listed above
2. Ensure text is readable and high contrast
3. Optimize file size for web use
4. Include source files for editability
5. Provide a brief description of intended use
