# OSSAfrica Logos

This directory contains official OSSAfrica logos in various formats and variations.

## Available Formats

- **SVG**: Vector format, scalable to any size (preferred for web and print)
- **PNG**: Raster format with transparency (for general use)
- **EPS**: Vector format for professional printing

## Logo Variations

Logos should be provided in multiple variations:

- **Primary Logo**: Full color version for standard use
- **Dark Version**: For use on dark backgrounds
- **Light Version**: For use on light backgrounds
- **Monochrome**: Single color version (black or white)
- **Icon Only**: Standalone icon/symbol without text

## Usage Guidelines

### Minimum Size
- Print: 1 inch wide minimum
- Digital: 120px wide minimum

### Clear Space
Maintain clear space around the logo equal to the height of the "O" in OSSAfrica (or equivalent element).

### Don'ts
- Don't stretch or distort the logo
- Don't change the colors (use provided variations)
- Don't add effects (shadows, outlines, gradients)
- Don't place on busy backgrounds without sufficient contrast

## File Naming Convention

Use this naming pattern:
```
ossafrica-logo-[variation]-[format].[ext]

Examples:
- ossafrica-logo-primary.svg
- ossafrica-logo-dark.png
- ossafrica-logo-monochrome-white.svg
- ossafrica-icon-only.svg
```

## Contributing

When contributing new logo variations:
1. Ensure consistency with existing brand identity
2. Provide both vector (SVG) and raster (PNG) formats
3. Include source files (AI, Sketch, Figma link, etc.)
4. Test on both light and dark backgrounds
5. Follow the naming convention above
